# 즐겨찾기 일정 백엔드 DTO / 컨트롤러 정리

작성일: 2026-06-11

## 1. 작업 목적

사용자가 즐겨찾기한 공고를 신청 일정 기준으로 묶어서 조회할 수 있도록
`GET /api/me/favorites/schedule` API를 추가했다.

이 API는 다음 정보를 한 번에 반환한다.

- 상단 요약 카운트
- 상태별 그룹 목록
- 캘린더 이벤트 목록
- 일정 확인 안내 문구

## 2. 컨트롤러

대상 파일:
`subscrition/src/main/java/com/ttait/subscription/notification/favorite/controller/FavoriteController.java`

### 추가된 엔드포인트

```http
GET /api/me/favorites/schedule
Authorization: Bearer {accessToken}
```

### 역할

- 현재 로그인한 사용자의 즐겨찾기 공고를 조회한다.
- 관리자 검수 완료 상태의 공고만 일정에 포함한다.
- 일정 상태를 계산해서 요약, 그룹, 캘린더 이벤트 형태로 반환한다.

### 컨트롤러 메서드

```java
@GetMapping("/schedule")
public ResponseEntity<FavoriteScheduleResponse> schedule() {
    return ResponseEntity.ok(favoriteScheduleService.getSchedule(CurrentUser.id()));
}
```

### 응답 타입

- `FavoriteScheduleResponse`

## 3. DTO 정리

DTO 위치:
`subscrition/src/main/java/com/ttait/subscription/notification/favorite/dto`

### 3-1. FavoriteScheduleResponse

파일:
`FavoriteScheduleResponse.java`

역할:
일정 조회 API의 최상위 응답 객체

필드:

- `summary`: 상단 요약 정보
- `groups`: 상태별 공고 그룹
- `calendarEvents`: 일정 캘린더 이벤트 목록
- `disclaimer`: 안내 문구

### 3-2. FavoriteScheduleSummaryResponse

파일:
`FavoriteScheduleSummaryResponse.java`

역할:
상단 카운트 요약

필드:

- `totalCount`: 전체 즐겨찾기 공고 수
- `dueTodayCount`: 오늘 마감 수
- `dueTomorrowCount`: 내일 마감 수
- `dueSoonCount`: 7일 이내 마감 수
- `openCount`: 현재 접수 가능한 공고 수
- `upcomingCount`: 접수 예정 수
- `dateUnknownCount`: 일정 확인 필요 수
- `closedCount`: 마감됨 수

주의:

- `dueSoonCount` 는 `DUE_TODAY`, `DUE_TOMORROW`, `DUE_SOON` 을 모두 포함한다.
- `openCount` 는 `DUE_TODAY`, `DUE_TOMORROW`, `DUE_SOON`, `OPEN` 을 모두 포함한다.

### 3-3. FavoriteScheduleGroupResponse

파일:
`FavoriteScheduleGroupResponse.java`

역할:
상태별 묶음 단위

필드:

- `key`: 상태 키
- `label`: 화면 표시용 라벨
- `items`: 해당 상태에 속한 공고 목록

예시 상태 그룹 순서:

- `DUE_TODAY`
- `DUE_TOMORROW`
- `DUE_SOON`
- `OPEN`
- `UPCOMING`
- `DATE_UNKNOWN`
- `CLOSED`

### 3-4. FavoriteScheduleItemResponse

파일:
`FavoriteScheduleItemResponse.java`

역할:
그룹 내부의 개별 공고 일정 정보

필드:

- `announcementId`: 공고 ID
- `noticeName`: 공고명
- `providerName`: 공급 기관명
- `noticeStatus`: 공고 상태
- `applicationStartDate`: 신청 시작일
- `applicationEndDate`: 신청 마감일
- `announcementDate`: 공고일
- `winnerAnnouncementDate`: 당첨자 발표일
- `favoritedAt`: 즐겨찾기 등록 시각
- `dDay`: 마감일까지 남은 일수
- `dDayLabel`: `D-day`, `D-7`, `마감` 같은 문자열
- `scheduleStatus`: 일정 상태 enum
- `statusMessage`: 상태 설명 문구
- `actionLabel`: 현재는 고정값 `공고 확인`

### 3-5. FavoriteCalendarEventResponse

파일:
`FavoriteCalendarEventResponse.java`

역할:
캘린더에 바로 표시할 수 있는 일정 이벤트

필드:

- `announcementId`: 공고 ID
- `eventType`: 이벤트 종류
- `date`: 일정 날짜
- `title`: 화면 표시용 제목
- `priority`: 우선순위

현재 사용하는 `eventType` 예시:

- `ANNOUNCEMENT_DATE`
- `APPLICATION_START`
- `APPLICATION_END`
- `WINNER_ANNOUNCEMENT`

현재 사용하는 `priority` 예시:

- `HIGH`
- `NORMAL`
- `LOW`

### 3-6. FavoriteScheduleStatus

파일:
`FavoriteScheduleStatus.java`

역할:
공고의 일정 상태를 분류하는 enum

값:

- `DUE_TODAY`: 오늘 마감
- `DUE_TOMORROW`: 내일 마감
- `DUE_SOON`: 7일 이내 마감
- `OPEN`: 접수 중
- `UPCOMING`: 접수 예정
- `DATE_UNKNOWN`: 일정 확인 필요
- `CLOSED`: 마감됨

## 4. 상태 계산 기준

상태 계산은 `FavoriteScheduleService` 에서 수행한다.

대상 파일:
`subscrition/src/main/java/com/ttait/subscription/notification/favorite/service/FavoriteScheduleService.java`

기준:

- 시작일 또는 마감일이 없으면 `DATE_UNKNOWN`
- 시작일이 마감일보다 뒤면 `DATE_UNKNOWN`
- 오늘이 마감일보다 뒤면 `CLOSED`
- 오늘이 시작일보다 앞이면 `UPCOMING`
- 마감일까지 0일 남으면 `DUE_TODAY`
- 마감일까지 1일 남으면 `DUE_TOMORROW`
- 마감일까지 7일 이하 남으면 `DUE_SOON`
- 그 외 접수 기간 내 공고는 `OPEN`

## 5. 실제 응답 확인 결과

Apidog 수동 테스트 기준으로 아래 형태까지 확인했다.

- `summary.totalCount = 1`
- `summary.dueSoonCount = 1`
- `summary.openCount = 1`
- `groups[DUE_SOON]` 에 테스트 공고 1건 포함
- `calendarEvents` 에 `APPLICATION_START`, `APPLICATION_END` 생성 확인

예시 응답 핵심:

```json
{
  "summary": {
    "totalCount": 1,
    "dueSoonCount": 1,
    "openCount": 1
  },
  "groups": [
    {
      "key": "DUE_SOON",
      "items": [
        {
          "announcementId": 5,
          "noticeName": "Apidog 일정 테스트 공고",
          "dDayLabel": "D-7",
          "scheduleStatus": "DUE_SOON"
        }
      ]
    }
  ],
  "calendarEvents": [
    {
      "announcementId": 5,
      "eventType": "APPLICATION_START"
    },
    {
      "announcementId": 5,
      "eventType": "APPLICATION_END"
    }
  ]
}
```
